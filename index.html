Use Cases

AI Research

  • Query local corpora without exposing sources.
  • Blend vector and lexical retrieval for precision.
  • Synthesize answers with citation-ready context.
$ nex query "protein folding mechanisms" ▶ Searching local corpus... ▶ Hybrid BM25 + vector match: 94.2% ▶ Synthesizing with Zayvora... ✓ Response ready [0 API calls]

Study & Learning

  • Run study sessions fully offline.
  • Track notes and revisions in local-first storage.
  • Build reusable learning loops with no SaaS lock-in.
$ studyos start --mode focused ▶ Loading local decks... ▶ Session timer: 45 min ▶ Prompting with Zayvora tutor mode... ✓ Session synced to local ledger

Developer Tools

  • Orchestrate CLI workflows with Orchade.
  • Chain local models for coding, review, and refactor.
  • Maintain full logs for audit and replay.
$ orchade run build-agent ▶ Task graph initialized ▶ Agent chain: lint → patch → verify ▶ Local sandbox execution... ✓ Build completed with zero remote calls

Local Agents

  • Deploy autonomous assistants in private environments.
  • Schedule periodic tasks with deterministic logs.
  • Control access with machine-local permissions.
$ zayvora agent deploy ops-watch ▶ Loading policy: local-only ▶ Attaching Nex memory index ▶ Enabling Orchade scheduler ✓ Agent active [scope: this machine]
